# Sistema de Estoque e Vendas

## Descrição

Sistema de linha de comando desenvolvido em Python para gerenciamento de estoque e vendas. O sistema permite cadastrar, editar, remover e consultar produtos, além de registrar vendas e gerar relatórios.

## Funcionalidades

* Cadastro de produtos
* Edição de produtos
* Remoção de produtos
* Busca por código
* Busca por nome
* Registro de vendas
* Listagem de produtos
* Filtro por categoria
* Relatório de estoque baixo
* Salvamento e carregamento de dados em JSON

## Estrutura do Projeto

* main.py -> Menu principal e fluxo do sistema
* produto.py -> Classe Produto
* estoque.py -> Operações de estoque
* arquivos.py -> Persistência de dados
* dados.json -> Armazenamento dos produtos

## Como Executar

1. Abra o projeto no VS Code.
2. Abra o terminal.
3. Entre na pasta do projeto.
4. Execute:

python main.py

## Tecnologias Utilizadas

* Python 3
* JSON
